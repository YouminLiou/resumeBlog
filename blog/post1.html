網站結構設計與語意化規劃

<header>
      <main>
      <footer>

整體以 section + id 控制 anchor navigation，避免未來引入 Router 或 SPA 架構時要重構 DOM 結構。

視覺系統：Glassmorphism + Neumorphism

  • 背景採 radial 深藍 + 透明玻璃模糊
  • 卡片區塊採用 Neumorphism 陰影設計
  • 字體採用 Roboto Mono
  • RWD 使用 CSS Flex + Clamp

元件模組化與樣式管理

樣式集中於 styles.css，使用 CSS 變數與 box-shadow token 模組化，利於轉移到 Sass / Tailwind 架構。

表單功能：支援 Google Form 後端串接

使用 Google Form 的 formResponse API，無需 JS 後端服務，簡潔穩定。

部署架構

  • Vercel 一鍵部署
  • 加入 SEO、OG、Twitter Meta
